## 紹介

SimplePaintは、線画から着色、陰影までAIが自動化するイラスト＆ウェブトゥーン制作ツールです。
ご自身のAPIキー（Gemini、Flux）を使用して、AI機能を無料でご利用いただけます。

---

## AI機能

### 1. 陰影生成 (AI Shading)

線画レイヤーから自動的にセルシェーディングを生成します。光源の方向と強度を自動分析し、自然な陰影を作成します。

**使用方法:**
1. 線画レイヤーを選択
2. 上部メニューから [AI] → [陰影生成] をクリック
3. オプション選択: ハードセルシェーディング / ソフトシェーディング / 3トーンセルシェーディング
4. 生成ボタンをクリック

### 2. ベース色生成 (AI Colorize)

線画に自動的にフラットカラーリングを適用します。キャラクターシートを参照して一貫した色を維持できます。

**使用方法:**
1. 線画レイヤーを選択
2. [AI] → [ベース色生成] を選択
3. (オプション) キャラクターシートファイルを指定
4. 自動着色を実行

### 3. キャラクターシート生成

参照画像からターンアラウンドキャラクターシートを自動生成します。正面、側面、背面、クローズアップの4つのビューを提供します。

**使用方法:**
1. [AI] → [キャラクターシート生成] を選択
2. 参照画像をアップロード (正面または3/4アングル推奨)
3. 生成するビューを選択 (正面/側面/背面/クローズアップ)
4. 生成を実行

### 4. 線画生成 (AI Line Art)

キャラクターシートを基に新しいポーズの線画を生成します。クリーン線画、自然なペン線、スケッチ風から選択可能です。

**使用方法:**
1. キャラクターシートをロード
2. [AI] → [線画生成] を選択
3. 希望のポーズプロンプトを入力 (例: "座っているポーズ")
4. 線画スタイルを選択
5. 生成を実行

### 5. 領域分離 (AI Segmentation)

画像を意味のある単位に自動分離し、それぞれ別のレイヤーにします。

**使用方法:**
1. 分離するレイヤーを選択
2. [AI] → [領域分離] を選択
3. 分離する要素を入力 (例: "人物、背景、小道具")
4. 自動分離を実行

### 6. 陰影色変換 (Shading Color Swap)

グレートーンの陰影を希望のカラーに変換して様々な雰囲気を演出できます。AI機能ではないため、APIキーは不要です。

**使用方法:**
1. 陰影レイヤーを選択
2. [AI] → [陰影色変換] を選択
3. 変換する色を選択 (カラーピッカー)
4. 適用ボタンをクリック

## インターフェース＆基本機能

### メニューバー

**File メニュー**
- **Save Project (Ctrl+S):** 現在の作業をプロジェクトファイルとして保存します。レイヤー構造がそのまま保持されます。
- **Open Project (Ctrl+O):** 保存されたプロジェクトファイルを開きます。
- **Export Image (Ctrl+E):** 現在のキャンバスを画像ファイル（PNGなど）にエクスポートします。
- **Import Image to Layer:** 外部画像ファイルを新しいレイヤーとしてインポートします。
- **Import PSD (Ctrl+Shift+O):** Photoshop PSDファイルをレイヤー構造そのまま読み込みます。
- **Export as PSD (Ctrl+Shift+S):** 現在の作業をPSDファイルとしてエクスポートします。
- **Exit (Ctrl+Q):** アプリケーションを終了します。

**Edit メニュー**
- **Undo (Ctrl+Z):** 最後の操作を元に戻します。
- **Redo (Ctrl+Y):** 元に戻した操作をやり直します。
- **Free Transform (Ctrl+T):** 選択されたレイヤーを自由変形します。サイズ変更、回転、移動が可能です。
- **Canvas Size:** キャンバスのサイズを変更します。
- **AI Variations:** AIを使用して現在の画像のバリエーションを生成します。
- **Clear Layer:** 現在選択されているレイヤーの内容をすべて消去します。

**View メニュー**
- **Zoom In (Ctrl++):** キャンバスを拡大します。
- **Zoom Out (Ctrl+-):** キャンバスを縮小します。
- **Reset Zoom (Ctrl+0):** キャンバスをデフォルトのズームレベル（100%）にリセットします。
- **Show Grid:** キャンバスのグリッド表示を切り替えます。

### ツールバー

- **Undo / Redo (Ctrl+Z / Ctrl+Y):** 操作を元に戻したりやり直したりします。
- **Save / Load:** 現在のキャンバスを画像ファイルで保存、または画像ファイルを読み込みます。
- **Color:** 描画に使用する色を選択します。デフォルト4色が提供され、クリックでカスタム色を選択できます。
- **Eraser:** 消しゴムモードのオン/オフを切り替えます。
- **Clear:** 現在選択されているレイヤーの内容をすべて消去します。
- **Brush Size:** ブラシ（ペン/消しゴム）のサイズを調整します。
- **Zoom:** キャンバスのズームレベルを調整します。

### レイヤーシステム

**レイヤー設定**
- **ブレンドモード:** レイヤーパネル上部のドロップダウンからブレンドモードを選択できます。14種対応。
- **不透明度 (Opacity):** スライダーでレイヤーの不透明度を0%～100%の間で調整します。

**レイヤー管理**
- **+Layer:** 新しい空のレイヤーを追加します。
- **+Folder:** レイヤーをグループ化できるフォルダを追加します。
- **Delete:** 選択されたレイヤーまたはフォルダを削除します。
- **Up / Down:** 選択されたレイヤーの順序を1段階上または下に移動します。
- **Show / Hide:** 選択されたレイヤーを非表示にしたり、再表示したりします。

**高度な機能**
- **+Mask:** 選択されたレイヤーにマスクを追加します。
- **-Mask:** 選択されたレイヤーのマスクを削除します。
- **Edit Mask:** マスク編集モードに入ります。消しゴムで領域を消すとその部分がマスクされます。完了したらEdit Maskを再度クリックして終了します。
- **Clip:** 選択されたレイヤーを下のレイヤーにクリッピングします。下のレイヤーで塗られた領域内でのみ現在のレイヤーが表示されます。

### ブレンドモード (14種)

| グループ | モード |
|---------|--------|
| 暗く | Normal, Darken, Multiply, Color Burn, Linear Burn |
| 明るく | Lighten, Screen, Color Dodge, Add (Linear Dodge) |
| コントラスト | Overlay, Soft Light, Hard Light |
| 差 | Difference, Exclusion |

## APIキー設定

SimplePaint Freeはご自身のAPIキーを使用します。AI機能を利用するには以下のAPIキーが必要です:

| API | 用途 | 発行元 |
|-----|------|--------|
| Flux (BFL) | 陰影生成、ベース色生成、線画生成 | [Black Forest Labs](https://api.bfl.ml/) |
| Gemini (Google) | キャラクターシート生成、領域分離、色分析 | [Google AI Studio](https://aistudio.google.com/) |

アプリ起動後、**設定 → API Settings**でキーを入力してください。
